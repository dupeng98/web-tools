# 前端工具集

一个简洁的 iOS 风格前端工具集合。

## 功能

- **JSON 在线解析** - 格式化、压缩和验证 JSON 数据
- **驼峰名转换** - 字符串命名格式转换（显示所有格式：驼峰、帕斯卡、下划线、横线、空格）
- **时间计算器** - 日期时间差计算（显示天、小时、分钟、秒）和时间加减操作（支持同时设置天、小时、分钟、秒）
- **随机生成数据** - 生成随机中文名、手机号、身份证号

## 使用

打开 `index.html` 开始使用，选择需要的工具。

## 文件结构

- `index.html` - 工具集首页
- `json-parser.html` - JSON 解析工具
- `camel-case-converter.html` - 命名转换工具
- `time-calculator.html` - 时间计算工具
- `random-data-generator.html` - 随机数据生成工具

## 技术栈

- HTML5
- CSS3 (iOS 风格设计)
- JavaScript (ES6+)
